## Reporting Security Issues

If you discover a security vulnerability in this portfolio or any of its projects, please email me directly at **harshith6754@gmail.com** instead of using the public issue tracker.

### Please include:
- Description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact
- Suggested fix (if you have one)

I will acknowledge your report within 48 hours and work with you to resolve the issue responsibly.

## Responsible Disclosure

Please give me reasonable time to fix the issue before public disclosure. Generally:
- Critical issues: 7 days
- High priority: 14 days
- Medium priority: 30 days

## Scope

This security policy applies to:
- The Harshith-OS portfolio website
- All custom security tools in this repository
- Demonstrations and POC code

## Out of Scope

- Third-party dependencies and libraries
- Vulnerabilities in external websites or services
